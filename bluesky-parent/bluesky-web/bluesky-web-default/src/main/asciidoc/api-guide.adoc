= 개요

[[개요-http-verbs]]
== HTTP verbs

RESTful 메모는 HTTP 동사 사용시 표준 HTTP 및 REST 규칙에 최대한 가깝게 노력합니다.

|===
| Verb | Usage

| `GET`
| 리소스를 검색하는 데 사용됩니다.

| `POST`
| 새 리소스를 만드는 데 사용됩니다.

| `PATCH`
| 부분 업데이트를 포함하여 기존 리소스를 업데이트하는 데 사용됩니다.

| `DELETE`
| 기존 자원을 삭제하는 데 사용됩니다.
|===

[[개요-http-status-codes]]
== HTTP status codes

RESTful notes tries to adhere as closely as possible to standard HTTP and REST conventions in its
use of HTTP status codes.

|===
| Status code | Usage

| `200 OK`
| 요청이 완료되었습니다.

| `201 Created`
| 새 리소스가 성공적으로 만들어졌습니다. 리소스의 URI는 응답의 Location헤더 에서 사용할 수 있습니다.
`Location` header

| `204 No Content`
| 기존 리소스에 대한 업데이트가 성공적으로 적용되었습니다.

| `400 Bad Request`
| 요청이 잘못되었습니다. 응답 본문에는 추가 정보를 제공하는 오류가 포함됩니다.

| `404 Not Found`
| 요청한 리소스가 존재하지 않습니다.
|===

[[개요-hypermedia]]
== Hypermedia

RESTful Notes uses hypermedia and resources include links to other resources in their
responses. Responses are in http://stateless.co/hal_specification.html[Hypertext Application
from resource to resource.
Language (HAL)] format. Links can be found beneath the `_links` key. Users of the API should
not create URIs themselves, instead they should use the above-described links to navigate
