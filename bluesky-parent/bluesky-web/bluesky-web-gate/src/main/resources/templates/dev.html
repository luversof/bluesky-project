FE 관련 개발 설정

npm 설정

`src/main/frontend` 위치에서 아래 명령 실행하면 FE 프로젝트 관련 설정이 설치됩니다. 
npm install

watch 사용

프로젝트 최상위 위치에 제공되는 `npm-watch.bat`를 실행하거나 
npm-watch.bat
`src/main/frontend` 위치에서 직접 실행하면 개발하는 template 소스의 tailwindcss class 변경이 실시간으로 반영됩니다. 
npm run watch
