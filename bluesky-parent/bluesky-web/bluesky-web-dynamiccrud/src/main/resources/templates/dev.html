BE 개발 관련 dom attribute 설정 정보

tailwindcss나 daisyui의 class 명명 규칙과 별개로 사용하기 위한 값들에 대한 목록
  • 검색 영역 관련
    • #searchArea : 검색 영역 지정
    • .searchInput : 검색 영역의 검색 input 지정
    • .resetButton : 검색의 reset 버튼
    • .searchButton : 검색의 search 버튼
  • 목록 영역 관련
    • #listArea : 목록 영역 지정
    • contentTable 관련
      • #content : content 영역 지정
      • #contentTable : content를 보여주는 테이블 지정
      • .contentData : list 목록의 input hidden 정보를 담고 있는 영역 지정
      • .contentDataCheck : list 목록의 checkbox 영역 지정
      • [name=contentDataCheck] : contentTable의 content 목록의 checkBox
      • .deleteButton : 삭제 버튼
      • .navButton : pagination의 링크 trigger 버튼
  • modal 관련
    • 공통 설정
      • css selector
        • #fragmentArea : fragment 호출 영역 (에러 발생시 에러를 표시되는 영역이기도 함)
        • #modal : modal의 id (현재까지 modal이 1종이라 별다른 이름을 만들지 않음)
      • HX-Trigger (요청 응답 이후 htmx trigger 처리를 위한 설정)
        • showModalForm : 생성 모달 뷰 응답
        • closeModalForm : 모달을 닫아야 하는 응답의 경우 (쓰이는 경우가 현재는 없음)
    • form modal 관련 (기본적인 CRUD modal)
      • css selector
        • #modalForm : modal의 form id. modal내 submit button의 대상 지정을 위해 사용
        • .copyDataButton : modal의 데이터 복사 버튼 (바닥 페이지의 선택한 데이터를 form으로 복사)
        • .copySearchButton : modal의 검색 복사 버튼
      • HX-Trigger (요청 응답 이후 htmx trigger 처리를 위한 설정)
        • showList : 페이지 호출 뷰 응답
        • createModalForm : 모달 뷰 응답
        • updateModalForm : 모달 뷰 응답
        • createModal : 생성 요청 응답
        • updateModal : 수정 요청 응답
        • deleteModal : 삭제 요청 응답
    • bulk form modal 관련 (import export)
      • css selector
        • #modalBulkForm : modal의 form id. modal내 submit buuton의 대상 지정을 위해 사용
        • .copyDataButton : modal의 데이터 복사 버튼(bulkform textarea의 데이터를 clipboard로 복사하는 역할)
      • HX-Trigger (요청 응답 이후 htmx trigger 처리를 위한 설정)
        • importModalBulkForm : 모달 뷰 응답
        • exportModalBulkForm : 모달 뷰 응답
        • importModalBulk : 생성 요청 응답
